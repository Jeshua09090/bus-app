# Soporte

Buses CR todavía está temprano, así que el soporte es principalmente discusión
del proyecto y triage de issues.

Usá GitHub Issues para:

- bugs en la app
- correcciones de rutas o paradas
- documentación de setup confusa
- resultados del planner que no coinciden con viajes reales

Incluí suficiente detalle para que otra persona pueda reproducir o verificar el
problema. Para reportes de rutas, lo más útil es origen, destino, hora
aproximada, ruta esperada y ruta que mostró la app.

Para reportes privados de seguridad, usá GitHub Security Advisories cuando sea
posible.
